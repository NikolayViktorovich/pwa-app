# Техническое задание: GameHub PWA  
## Описание проекта  
PWA-приложение с мини-играми (Крестики-нолики, Змейка, Аркада про Кабана), доступное офлайн.  

## Инструменты разработки  
- **Фреймворк**: Next.js (App Router, TypeScript)  
- **Стили**: Tailwind CSS + Shadcn/ui  
- **Анимации**: Framer Motion  
- **PWA**: Workbox, Web Push API  
- **Деплой**: Vercel  

## Этапы разработки  
1. **Спринт 1**: Постановка задачи и настройка репозитория.  
2. **Спринт 2**: Базовый проект + первая игра (Крестики-нолики).  
3. **Спринт 3**: Вторая игра (Змейкв) + PWA-уведомления.  
4. **Спринт 4**: Третья игра (Аркада) + деплой.  

## Функционал  
- Главная страница со списком игр.  
- Offline-режим и кеширование.  
- Push-уведомления о победах.  
- Адаптивный дизайн.
