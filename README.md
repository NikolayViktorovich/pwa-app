# REDD-PWA — Аркада "МЕГА КАБАН: БИТВА ЗА БУРГЕРЫ"

Приложение с динамичной мини-игрой "МЕГА КАБАН: БИТВА ЗА БУРГЕРЫ", разработанное как **Progressive Web App (PWA)** с оптимизированным геймплеем для мобильных и десктопных устройств.
---

## **Структура проекта**

### **Этапы разработки**
1. **Спринт 1**: Постановка задачи и настройка репозитория
2. **Спринт 2**: Разработка игрового ядра и механик
3. **Спринт 3**: PWA-разработка и оптимизация
4. **Спринт 4**: Финальная полировка и деплой

### **Стек технологий**
- **Фреймворк**: Next.js (App Router, TypeScript)
- **Стилизация**: Tailwind CSS + Shadcn/ui
- **Анимации**: Framer Motion
- **PWA-функции**: Workbox, Web Push API
- **Деплой**: Vercel

## **Цели проекта**
1. Создать увлекательную аркадную игру с простым, но аддиктивным геймплеем
2. Реализовать полноценный PWA с поддержкой всех современных функций
3. Обеспечить высокую производительность на всех типах устройств
4. Предоставить пользователям возможность играть офлайн и получать уведомления
5. Создать масштабируемую архитектуру для будущих обновлений и расширений
