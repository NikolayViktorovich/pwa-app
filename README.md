# GameHub PWA  
Приложение с коллекцией мини-игр, разработанное как **Progressive Web App (PWA)**.  
- **Тип проекта**: Веб-приложение с возможностью установки на устройство (PWA).  
- **Доступно на**: Мобильных и десктоп-устройствах.
